# Sea Level Trends & Storm Events in Miami-Dade, FL
A narrative-driven visualization tool that establishes statistical associations between rising sea levels and extreme weather events in Miami-Dade County.

The live interactive dashboard is available at: [https://tayibac.github.io/miami-dade-sea-level-trends/](https://tayibac.github.io/miami-dade-sea-level-trends/)

## Project Overview
This project integrates environmental data to analyze coastal vulnerability. It correlates historical sea-level trends with documented storm events to provide a comprehensive view of climate impact, contextualized by event narratives.

## Features
- Statistical correlation between sea-level rise and storm frequency.
- Data integration of multi-source NOAA datasets.
- Qualitative context through integrated news reports and public event narratives.
- Filtering for specific events: Coastal Flood, Flash Flood, Hurricane, Storm Surge, and Tsunami.

## Data Sources
- Storm Data: NOAA NCDC Storm Events Database
- Sea Level Trends: NOAA Tides and Currents
